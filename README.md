# ✈ AirportMS — Airport Management System

Sistema di gestione aeroportuale completo con GUI PyQt5.

## Avvio rapido
```bash
pip install PyQt5 pyttsx3
python main.py
```

## Moduli inclusi
- 🏠 Dashboard con statistiche in tempo reale
- ✈ Gestione Voli (CRUD + cambio stato + assegnazione gate casuale)
- 👤 Gestione Passeggeri
- 🚪 Gestione Gate (mappa visiva + lista)
- 🔒 Controlli di Sicurezza
- 📺 FIDS Kiosk full-screen (display partenze/arrivi)
- 🎫 Boarding & Verifica Biglietti + TTS annunci
- 👷 Gestione Personale
- ⏱ Timbro Cartellino (Clock IN/OUT)
- 📊 Report, Log e Esportazione CSV

## Compilazione
```bash
pip install py2exe
python setup.py py2exe
```
Vedi **GUIDA.txt** per istruzioni dettagliate.
